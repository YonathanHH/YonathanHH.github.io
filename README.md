# Yonathan Hary Hutagalung — Portfolio Website

Bloomberg Terminal-themed personal portfolio website.

## Pages
- `index.html` — Homepage: Introduction, featured projects, skills overview, experience timeline
- `cv.html` — Full CV with sticky sidebar navigation
- `portfolio.html` — Complete project list with category filters and GitHub links

## Deployment (GitHub Pages)
1. Create a repo named `YonathanHH.github.io` (or any repo name)
2. Upload all files to the root of the `main` branch
3. Go to **Settings → Pages → Deploy from branch → main → / (root)**
4. Your site will be live at `https://YonathanHH.github.io`
